## Logo Guidelines

- Use high-quality PNG files (transparent background preferred)
- Minimum size: 200x200px
- Official university logos only
- Maintain consistent styling
- Ensure good readability at all sizes

## Adding Logos

1. Download official logos from university websites
2. Resize to appropriate dimensions
3. Place in this folder
4. Update the logoPath in universities array in page.tsx

The website will automatically display the logos once they're added to this folder.
